

# Python Data Engineering Projects

This repository contains beginner-level **data engineering projects** implemented using Python.  
The projects demonstrate practical skills used in data pipelines, data extraction, and database management.

## Projects Included

### 1. ETL Pipeline
A Python ETL pipeline that extracts data from CSV, JSON, and XML files, transforms the data, and loads it into a final dataset.

Technologies:
- Python
- Pandas
- XML ElementTree

---

### 2. Web Scraping Project
A web scraping script that extracts the top 50 highest ranked films from a webpage and stores the data in CSV and SQLite database.

Technologies:
- Python
- BeautifulSoup
- Requests
- Pandas

---

### 3. SQLite Database Project
A project demonstrating database interaction using Python and SQLite. Data is loaded from a CSV file, queried using SQL, and updated with new records.

Technologies:
- Python
- SQLite
- SQL
- Pandas

---

## Skills Demonstrated

- ETL pipeline development
- Web scraping
- HTML parsing
- Data transformation
- SQL database operations
- Data automation using Python
